🚀 stellaris


stellar is, interactive web application that brings space exploration to your browser. Built with a sleek, Cyberpunk Terminal aesthetic, this platform allows users to explore a 3D solar system and discover fascinating data about our universe.

🌟 Project Overview
This project was developed as a comprehensive solution for the Codeveda internship program, covering three levels of technical progression: from fundamental UI development to advanced 3D visualization and API integration.

🛠️ Tech Stack
Frontend: React.js, TypeScript

3D Engine: Three.js, React-Three-Fiber

Styling: Tailwind CSS

Animations: Framer Motion

Data: NASA APIs

Deployment: Vercel

✨ Key Features
Interactive 3D Solar System: A fully navigable 3D environment featuring all 8 planets with realistic orbital paths.

Bento UI Design: A modern, clean, and responsive layout inspired by terminal dashboards.

Real-time Data: Integration with NASA APIs to fetch cosmic imagery and data.

High Performance: Optimized WebGL rendering to ensure smooth interaction across different devices.

Cyberpunk Aesthetic: A dedicated visual style featuring neon accents, dark gradients, and a immersive command center feel.
